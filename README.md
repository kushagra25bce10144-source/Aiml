## 🔍 Overview

This project implements a machine learning model to predict movie ratings (1-10) using features such as:
- Budget
- Runtime
- Genre
- Lead actor rating
- Release year

The model helps production companies estimate audience reception before release, aiding in investment decisions and marketing strategies.

## ✨ Features

- **Data Preprocessing**: Handles categorical variables and feature scaling
- **Random Forest Regression**: Robust ensemble learning algorithm
- **Feature Importance Analysis**: Identifies key rating influencers
- **Prediction Interface**: Easy-to-use prediction for new movies
- **Visualization**: Feature importance plots
